# Электронная таблица
Упрощенный аналог листа таблицы Microsoft Excel или Google Sheets. Ячейки таблицы могут содержать текст или формулы. Формулы, как и в существующих решениях, могут содержать индексы ячеек.
## Стек технологий
С++17, CMake, ANTRL.
